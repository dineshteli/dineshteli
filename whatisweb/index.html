What is website ?

A website (also written as web site) is a collection of web pages and related content that is identified by a common domain name and published on at least one web server. Notable examples are wikipedia.org, google.com, and amazon.com.

What are the types of webiste ?

1. Static websites

Static webistes are informative websites which are designed to provide information.

2. Dynamic websites

Dynamic webistes are used to store and access data from database.

What is database ?

A database is an organized collection of data, generally stored and accessed electronically from a computer system. Where databases are more complex they are often developed using formal design and modeling techniques.

What is domain name ?

A domain name is an identification string that defines a realm of administrative autonomy, authority or control within the Internet.

What is web hosting ?

A web hosting service is a type of Internet hosting service that allows individuals and organizations to make their website accessible via the World Wide Web.

What is SSL ?

Transport Layer Security, and its now-deprecated predecessor, Secure Sockets Layer, are cryptographic protocols designed to provide communications security over a computer network.

What is SEO ?

Search engine optimization is the process of improving the quality and quantity of website traffic to a website or a web page from search engines. SEO targets unpaid traffic rather than direct traffic or paid traffic.
